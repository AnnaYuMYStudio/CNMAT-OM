GET-COMBI

[GENERIC-FUNCTION]
CNMAT-OM
Function Reference
ARGUMENTS:
  - dur-space list of allowed durations [default = (1 2 3 4 5 6 8 10 12 15 20 24 30 40 60 120)]
  - sum sum of durations [default = 120]
  - num number of elements  
&optional
  
  - mode allow repeated elements? [default = 0]

Description:

Computes and returns the list of all combinations of <num> elements in <dur-space> which sum up to <sum>.
<mode> (0, 1 or 2) determines whether or not the list must remain ordered and if repetitions are allowed in the results.

<num> can be a signle number or a list of integers. If <num> = NIL allnumber of elements will be searched.
